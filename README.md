# Weather App

This is a simple and interactive weather app that fetches **real-time weather data** from the OpenWeatherMap API.  
It displays **temperature, humidity, wind speed**, and updates **weather icons** dynamically based on current conditions.  

## Live Demo

You can try it live here: [Weather App Live](https://khalidzaid8.github.io/Weather-App/)


## How to Use

1. Open the app in a browser.  
2. Type a city name in the search box and press the search button.  
3. Weather information for that city will be displayed.

## Tech Used

- HTML, CSS, JavaScript  
- OpenWeatherMap API
